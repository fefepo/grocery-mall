# 식자재 쇼핑몰 프로젝트


## Overview

기존 쇼핑몰의 기능을 기반으로, 사용자 편의성을 높이기 위한 추가 기능(찜 세트 묶음, 레시피 검색)을 구현하였습니다.  
---

## 개발환경
- React
- Firebase

---

## 담당 역할
- Firebase를 활용한 사용자 인증 및 권한 관리 (로그인, 회원가입 등) + 상품 데이터 관리 (이미지, 가격, 수량)
- 사용자 인터페이스(UI) 설계 및 구현
- 카카오페이 API를 활용한 QR 결제 시뮬레이션 기능 구현

---

## 주요 기능
- 로그인 / 회원가입 / 로그아웃 / 회원탈퇴
- 마이페이지
- 검색 및 카테고리 필터링
- 상품 상세 페이지
- 장바구니
- 상품 관리(관리자 화면)
- 카카오 qr 결제 기능
- 찜 세트 묶음 장바구니 기능
- 레시피 검색 기능
